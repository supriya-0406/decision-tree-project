# Decision Tree Classification Project

## Description
This project predicts customer purchase behavior using a Decision Tree Classifier.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the model:
   python decision_tree_model.py

## Output
- Accuracy score
- Confusion matrix
- Decision tree visualization
